Welcome to CyberGuard IR

AI-powered Incident Response Assistant

CyberGuard IR uses the Anthropic Claude API to provide expert-level incident response guidance. You'll need a free API key to get started — it takes about 2 minutes to set up.

Why do I need this?
✓ Your key is saved only in your browser — never on any server
✓ API calls go directly to Anthropic — no middleman
✓ New accounts get free credits to start with
✓ Typical IR session costs cents, not dollars

Enter your API key

Find your key at console.anthropic.com → API Keys. It starts with sk-ant-

How to get your key
  1. Go to console.anthropic.com and sign up (free)
  2. Click API Keys in the left sidebar
  3. Click Create Key and copy it
  4. Paste it above and click Save

You're all set!

Your API key is saved. CyberGuard IR is ready to assist with incident response. Open a new incident from the left panel to get started.

Quick tips
• Fill in the New Incident form on the left to begin
• AI guidance auto-loads for each IR phase
• Use Quick Actions buttons for common tasks
• Export your incident log anytime as a report
